# Regridder API Reference

::: xregrid.Regridder

The `Regridder` is the main class for performing regridding operations in XRegrid. It provides an optimized interface to ESMF (Earth System Modeling Framework) for interpolating data between different grids.

For high-level guides and examples, see the [Quick Start Guide](../user-guide/quickstart.md) and the [Examples Gallery](../examples/index.md).
